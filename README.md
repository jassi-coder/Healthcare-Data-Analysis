# 🏥 Healthcare Data Analysis

## 📊 Project Overview
This project focuses on analyzing a healthcare dataset using Python.  
The goal is to extract meaningful insights about patient demographics, medical conditions, and billing patterns.

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## 🔍 Key Analysis Performed
- Data Cleaning and Preprocessing  
- Age Group Analysis  
- Gender-based Analysis  
- Medical Condition Analysis  
- Admission Type Analysis  
- Billing Analysis  

---

## 📈 Key Insights
- Most patients belong to the **65+ age group**  
- Around **33.6% admissions are emergency cases**  
- Billing amount increases with age  
- Billing is relatively consistent across different medical conditions
